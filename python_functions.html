Как задается функция

Определение: Функция в Python задается с помощью ключевого слова def.

Синтаксис:


def имя_функции(параметры):  # создание функции
    # тело функции
    return результат

Когда использовать: Когда нужно повторно использовать блок кода в разных частях программы.

 

def add(a, b):  
    return a + b
      

result = add(3, 4)  # Результат: 7
Аргументы и параметры

Аргументы: Значения, передаваемые функции при вызове.
Параметры: Переменные, указанные в определении функции для приема значений.


def greet(name):
    print(f"Hello, {name}!")

Когда использовать: Когда нужно передавать данные в функцию для обработки.

 

def greet(name):  
    print(f"Hello, {name}!")
      

greet("Alice")  # Вывод: Hello, Alice!
Типы аргументов
  • Позиционные: Передаются в порядке, в котором они указаны.
  • Именованные: Передаются с указанием имени параметра.
  • *args: Для произвольного количества позиционных аргументов.
  • **kwargs: Для произвольного количества именованных аргументов.

def example(a, b=2, *args, **kwargs):
    print(a, b, args, kwargs)

Когда использовать: Когда необходимо передавать переменное количество данных в функцию.

 

example(1, 3, 4, 5, x=6, y=7)
      

# Вывод: 1 3 (4, 5) {'x': 6, 'y': 7}
Функции высшего порядка

Определение: Функции, которые принимают другие функции в качестве аргументов или возвращают их.

Синтаксис:


def apply_function(func, value):
    return func(value)

Когда использовать: Для повышения гибкости и функциональности кода, например, при работе с коллекциями.

 

def apply_function(func, value):
    return func(value)
      

def double(x):
    return x * 2

print(apply_function(double, 5))  # Результат: 10
Декораторы

Определение: Функции, которые изменяют поведение других функций.

Синтаксис:


def decorator(func):
    def wrapper(*args, **kwargs):
        print("Вызов функции")
        result = func(*args, **kwargs)
        print("Функция выполнена")
        return result
    return wrapper

Когда использовать: Для добавления дополнительного поведения к функциям, например, логирования, измерения времени выполнения, кэширования.

 

@decorator
def say_hello():
    print("Привет!")
      

say_hello()  
# Вывод:
# Вызов функции
# Привет!
# Функция выполнена
Генераторы

Определение: Функции, которые возвращают итераторы с помощью yield, позволяя возвращать данные по одному.

Синтаксис:


def fibonacci():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

Когда использовать: Для работы с большими данными или бесконечными последовательностями, где данные не нужно загружать все сразу в память.

 

def fibonacci():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b
      

fib = fibonacci()
for _ in range(5):
    print(next(fib))  # Вывод: 0 1 1 2 3
Рекурсивные функции

Определение: Функции, которые вызывают сами себя в своем теле.

Синтаксис:


def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)

Когда использовать: Для решения задач, где результат зависит от меньших версий самой задачи, например, вычисление факториала или обход дерева.

 

def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)
      

print(factorial(5))  # Результат: 120
Лямбда-функции

Определение: Однострочные анонимные функции, которые позволяют быстро определить функционал без создания полноценной функции.

Синтаксис:

lambda параметры: выражение

Когда использовать:

  • Для создания временных функций внутри других операций.
  • Для работы с функциями высшего порядка (map, filter, sorted).
  • Когда не нужно многократное использование функции.
 

  # Лямбда без имени
  square = lambda x: x**2
  
  # Лямбда с использованием _
  increment = lambda x, _: x + 1
  
  # Лямбда с разными коллекциями
  list_processor = lambda lst: [x**2 for x in lst]
  dict_processor = lambda dct: {k: v**2 for k, v in dct.items()}
  
  # Использование с условием
  compare = lambda x, y: 'Equal' if x == y else 'Greater' if x > y else 'Lesser'
      

  # Вызов без имени
  print(square(4))  # Результат: 16
  
  # Вызов с игнорируемым аргументом
  print(increment(5, 10))  # Результат: 6
  
  # Вызов с обработкой списка
  print(list_processor([1, 2, 3]))  # Результат: [1, 4, 9]
  
  # Вызов с обработкой словаря
  print(dict_processor({'a': 2, 'b': 3}))  # Результат: {'a': 4, 'b': 9}
  
  # Сравнение чисел
  print(compare(3, 5))  # Результат: Lesser
